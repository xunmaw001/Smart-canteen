# Smart-canteen
基于 Spring Boot + Vue 的智慧食堂系统，为食堂管理和用户就餐带来便捷体验。它集成了菜品展示、在线订餐、库存管理、订单处理等功能。Vue 构建的前端界面美观，方便用户操作，Spring Boot 搭建的后端提供稳定服务，保障数据的安全存储与高效处理。支持在线支付、评价反馈，提高食堂服务效率和质量，让用户吃得更舒心，食堂管理更高效。
